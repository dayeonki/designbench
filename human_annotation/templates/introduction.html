Graphic Design Annotation Task

Thank you for participating in this study aimed at validating 20 user queries and user-provided images for a graphic design generation task.

The goal of the graphic design generation task is to create graphic designs (e.g., a postcard) based on information from the user query.

The user query reflects the user's request for creating one graphic design, including design choices such as background color, text, and images, where applicable. The user-provided images are those the user intends to incorporate into their design.

You will be asked to perform the following task:

  1. Answer if the user query and the design choices are well-aligned at a 5-point Likert scale.
  2. Rank the three images provided based on how well they fit the user query. Assign a rank from 1 (best fit) to 3 (least fit).

You will be provided GPT's assessment and its confidence level to aid your decision making.

This annotation is expected to take 30-40 minutes in total.
